Welcome to Algorithms

How to use the algorithms:
--------------------------

Clone the repository:
```
git clone git@github.com:geftimov/Algorithms.git
```
Go in to the folder:
```
cd Algorithms
```
Build it with Maven:
```
mvn clean install
```
